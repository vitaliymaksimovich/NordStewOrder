**Описание**

Приложение позволяет пользователю собрать блюдо “Нордское рагу” с различными добавками, оформить заказ и сохранить его в историю.

Проект реализован с использованием:

Java 23, 
Maven, 
Swing (GUI), 
паттерна Decorator.


**Автор**

Черкасов Виталий, группа Б24-912


**Функциональность**

✔ Выбор добавок к блюду (максимум 3)
✔ Динамический расчёт итоговой стоимости
✔ Подтверждение заказа через диалоговое окно
✔ Сохранение заказов в файл
✔ Просмотр истории заказов
✔ Улучшенный графический интерфейс


**Использование паттерна Decorator**

В проекте реализован паттерн Decorator:

Dish — компонент (интерфейс);

NordStew — базовое блюдо (ConcreteComponent);

DishDecorator — базовый декоратор.


Добавки:

FireSauceDecorator;

DoubleMeatDecorator;

BerriesDecorator;

FlatbreadDecorator.


Каждая добавка динамически расширяет:

описание блюда,
стоимость,
Интерфейс.


**Особенности реализации**

логика ограничения добавок реализована в GUI

модель не содержит условий (if) для добавок

используется композиция объектов вместо наследования

соблюдено разделение:

UI

бизнес-логика

работа с файлами
